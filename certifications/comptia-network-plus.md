# CompTIA Network+

**Issuer:** CompTIA  
**Completed:** August 2021  
**Status:** Active  
**Exam Code:** N10-008

## What I Learned

- Networking concepts: OSI model in depth, TCP/IP stack, subnetting, IPv6, NAT, and routing protocols
- Network infrastructure: switch configuration, VLANs, spanning tree, wireless networking standards and security
- Network operations: monitoring tools, packet analysis, network documentation, change management
- Network security: firewalls, IDS/IPS, VPN types, AAA (Authentication, Authorization, Accounting)
- Network troubleshooting: methodology, tools (ping, traceroute, nslookup, netstat), common failure scenarios
- Cloud and virtualization: how networking applies in cloud environments and SDN concepts

## How I Use This at Work

The Network+ directly underpins my infrastructure work. At **F12.net**, VLAN design, switch deployment, and VPN configuration on WatchGuard and SonicWall platforms all draw on the concepts validated by this certification. At **ESW IT**, network troubleshooting, switch deployment, and SonicWall firewall configuration are regular tasks.

The Network+ was also the stepping stone toward the **WatchGuard PCNSE** I am currently pursuing — it validated the fundamentals that the vendor-specific cert now builds on.
