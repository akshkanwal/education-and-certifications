# Microsoft 365 Certified: Modern Desktop Administrator Associate

**Issuer:** Microsoft  
**Completed:** June 2022  
**Status:** Active  
**Exam Codes:** MD-100 (Windows Client) and MD-101 (Managing Modern Desktops)

## What I Learned

- Windows 10/11 deployment: Autopilot, Windows Deployment Services, MDT, in-place upgrades
- Device management with Microsoft Intune: enrollment methods, compliance policies, configuration profiles
- Microsoft Entra ID (Azure AD): join types, hybrid join, conditional access, MFA policies
- Application management: MSIX packaging, Win32 app deployment via Intune, Microsoft Store for Business
- Identity and access management: user and group management, RBAC, PIM concepts
- Update management: Windows Update for Business, update rings, feature update deferral
- Endpoint security: Microsoft Defender configuration, BitLocker, Windows Firewall management via Intune
