# CompTIA A+

**Issuer:** CompTIA  
**Completed:** February 2021  
**Status:** Active  
**Exam Codes:** 220-1001 (Core 1) and 220-1002 (Core 2)

## What I Learned

- Hardware: CPU architecture, RAM types, storage technologies (HDD, SSD, NVMe), motherboard components, power supplies, and peripheral interfaces
- Networking: basic TCP/IP, wireless standards, network troubleshooting, soho router configuration
- Operating systems: Windows 10, macOS, Linux — installation, configuration, and troubleshooting
- Security fundamentals: malware types, social engineering, best practices for securing workstations
- Troubleshooting methodology: a structured, documented approach to diagnosing both hardware and software issues
- Mobile device management basics and cloud computing fundamentals
- Virtualization concepts and basic cloud infrastructure

## How I Use This at Work

The A+ is the baseline for everything IT. At **Microserve**, the hardware and imaging knowledge was applied daily — configuring systems for enterprise clients, performing QA checks, and troubleshooting configuration issues before shipment. At **F12.net** and **ESW IT**, hardware troubleshooting for end users and the structured support methodology from the A+ remains the backbone of how I approach Tier 1 and Tier 2 tickets.

The security and OS knowledge from the A+ also connects to broader work including endpoint hardening, software deployment, and user access management.
