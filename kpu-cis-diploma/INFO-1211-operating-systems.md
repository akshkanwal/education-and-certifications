# INFO-1211 — Operating Systems

**Program:** KPU Computer Information Systems Diploma  
**Topic Area:** OS Internals & Administration

## What This Course Covered

- Windows and Linux OS architecture: kernel, user space, system calls
- Process management: creation, scheduling, states, and termination
- Memory management: virtual memory, paging, segmentation
- File system structures: NTFS, FAT32, ext4 — permissions, paths, and management
- User account management, access control, and privilege levels
- System configuration, registry management (Windows), and system services

## Key Skills Developed

- Managing processes and services at both GUI and command-line level
- Understanding what happens under the hood when an OS exhibits slow or erratic behaviour
- Configuring user accounts, permissions, and access levels appropriately
- Reading system logs and correlating events to diagnose OS-level issues

## How This Applies to My Career

Operating system knowledge is the core of Tier 2 and Tier 3 support. At **F12.net** and **ESW IT**, resolving escalated tickets regularly requires going beyond the surface symptom — looking at running processes, service states, event logs, and file system permissions to find the actual cause.

Active Directory administration at both organizations builds directly on the user account and access control concepts from this course. Understanding how Windows manages identities, permissions, and policy at the OS level makes Group Policy configuration and troubleshooting much more intuitive.
