# INFO-1214 — Discrete Mathematics for IT

**Program:** KPU Computer Information Systems Diploma  
**Topic Area:** Mathematical Foundations of Computing

## What This Course Covered

- Number systems: binary, octal, hexadecimal — conversion between all bases
- Boolean algebra: AND, OR, NOT, XOR — truth tables and logic simplification
- Set theory: unions, intersections, complements, Venn diagrams
- Logic and propositional calculus: if-then statements, logical equivalences
- Basic graph theory: nodes, edges, paths — applied to network topology concepts
- Combinatorics: permutations and combinations

## Key Skills Developed

- Converting between number systems without a calculator
- Understanding how binary arithmetic underlies IP addressing and subnetting
- Applying boolean logic to understand how firewall rules and access control lists evaluate conditions
- Reading and constructing logical expressions in programming and scripting

## How This Applies to My Career

This course is foundational in ways that are not always visible but are constantly present. Subnetting is binary math — understanding that a /24 subnet mask is 11111111.11111111.11111111.00000000 in binary makes IP addressing intuitive rather than memorized.

Boolean logic underlies firewall ACL evaluation, conditional logic in PowerShell scripts, and the way Group Policy applies based on OU membership and filter conditions. The graph theory basics connect directly to understanding network topology and routing decisions. It is the kind of course whose value only becomes clear once you start working in technical environments.
