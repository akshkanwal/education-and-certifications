# INFO-2416 — Server Operating Systems

**Program:** KPU Computer Information Systems Diploma  
**Topic Area:** Windows Server Administration

## What This Course Covered

- Windows Server installation, roles, and features
- Active Directory Domain Services (AD DS): forest, domain, OU structure, schema
- User and group management: account policies, group types, membership strategies
- DNS configuration: forward and reverse lookup zones, records, scavenging
- DHCP configuration: scopes, reservations, options, failover
- Group Policy: GPO creation, linking, filtering, precedence (LSDOU), troubleshooting
- File and print services: NTFS permissions, share permissions, DFS namespaces
- Remote management: RDP, WinRM, PowerShell remoting

## Key Skills Developed

- Building and managing an Active Directory environment from scratch
- Designing OU structures that support clean Group Policy application
- Troubleshooting GPO application using gpresult and RSOP
- Administering DNS and DHCP in production Windows environments

## How This Applies to My Career

This course is the foundation of the majority of my day-to-day IT work. Active Directory administration, Group Policy management, DNS and DHCP oversight — these are constant tasks at both **F12.net** and **ESW IT** across MSP client environments.

Working across multiple client AD environments in an MSP context requires exactly the skills this course built: understanding how AD is structured, how Group Policy flows and where it can break, and how to make changes safely without disrupting production. The move into **Azure AD / Entra ID** and **Intune** at ESW IT builds directly on this on-premises foundation, extending the same identity and policy management concepts to the cloud.
