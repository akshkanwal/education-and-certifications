# INFO-1212 — Networking Technologies 1

**Program:** KPU Computer Information Systems Diploma  
**Topic Area:** Networking Fundamentals

## What This Course Covered

- Network topologies: star, bus, ring, mesh
- The OSI model — all 7 layers with protocols and devices at each layer
- TCP/IP fundamentals: IPv4 addressing, classes, subnetting, CIDR notation
- Ethernet and switching: MAC addresses, frames, collision domains, broadcast domains
- Routing basics: default gateways, static routing, routing tables
- Network cabling: Cat5e, Cat6, fibre — standards and use cases

## Key Skills Developed

- Subnetting by hand and with tools — calculating network addresses, broadcast addresses, host ranges
- Identifying which layer of the OSI model a problem belongs to and troubleshooting from there
- Configuring basic switch settings and understanding VLAN concepts
- Reading and interpreting network diagrams

## How This Applies to My Career

This is one of the most directly applied courses in the entire program. At **F12.net** as an Infrastructure Specialist, I designed and implemented LAN/WAN environments that required accurate subnetting, VLAN segmentation, and structured cabling. At **ESW IT**, I handle switch deployments and network troubleshooting regularly.

The OSI model mental model is not just an exam concept — it is a genuine troubleshooting tool. When a connectivity issue comes in, working from Layer 1 (physical — is the cable plugged in, is the port up?) through to Layer 7 (application — is DNS resolving, is the firewall passing the traffic?) gives a systematic path to resolution every time.
