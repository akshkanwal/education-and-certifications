# INFO-2411 — Computer Security

**Program:** KPU Computer Information Systems Diploma  
**Topic Area:** Information Security Principles & Practice

## What This Course Covered

- CIA Triad: Confidentiality, Integrity, Availability — the core framework for all security decisions
- Network security fundamentals: perimeter defence, DMZ architecture, segmentation
- Encryption: symmetric (AES, DES), asymmetric (RSA, PKI), hashing (SHA, MD5)
- Firewall types: packet filtering, stateful inspection, application-layer (proxy)
- Vulnerability management: CVEs, patch management lifecycle, risk assessment
- Authentication and access control: MFA, RBAC, least privilege principle
- Social engineering and phishing awareness — human factors in security

## Key Skills Developed

- Applying the CIA triad as a decision framework when evaluating security controls
- Configuring firewalls with a deny-by-default, allow-by-exception philosophy
- Understanding how encryption secures data in transit and at rest
- Assessing risk and prioritizing remediation based on impact and likelihood

## How This Applies to My Career

Security is woven into every aspect of modern IT work. At **ESW IT**, I work with **SonicWall** firewall configuration where the principles from this course — stateful inspection, zone-based policies, IPS/IDS tuning — are directly applied. Security subscription configuration (content filtering, GAV, IPS) maps directly to the threat model concepts from INFO-2411.

The **KnowBe4** security awareness training management, **EPDR** (endpoint detection and response) oversight, and **MFA** enforcement work I do at ESW IT are all grounded in the human and technical security principles this course established.
