# INFO-2313 — Program Structure & Design 2

**Program:** KPU Computer Information Systems Diploma  
**Topic Area:** Advanced Programming & Scripting

## What This Course Covered

- Advanced modular programming: functions, parameters, return values, recursion
- Object-oriented concepts: classes, objects, inheritance, encapsulation
- File I/O: reading from and writing to files programmatically
- Error handling and debugging techniques
- Code documentation and maintainability standards
- Introduction to scripting for automation use cases

## Key Skills Developed

- Writing reusable, modular code that can be maintained and extended
- Debugging systematically — reading error messages, tracing execution, isolating faults
- Structuring scripts to handle edge cases and errors gracefully
- Documenting code so that other people (or future self) can understand and modify it

## How This Applies to My Career

The advanced programming skills from this course directly inform my **PowerShell scripting** for IT automation. Writing a script to bulk-onboard users in Active Directory, export license assignments from M365, or automate a deployment process requires the same modular thinking — break it into functions, handle errors, log output, document the script.

At **ESW IT**, scripting reduces repetitive manual work and makes complex tasks repeatable and auditable. The debugging methodology from this course also applies directly to troubleshooting scripts and automation pipelines that others have built.
