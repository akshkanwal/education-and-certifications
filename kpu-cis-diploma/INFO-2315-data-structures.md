# INFO-2315 — Data Structures

**Program:** KPU Computer Information Systems Diploma  
**Topic Area:** Algorithms & Data Organization

## What This Course Covered

- Arrays and multi-dimensional arrays: indexing, traversal, use cases
- Linked lists: singly and doubly linked, insertion and deletion operations
- Stacks (LIFO) and queues (FIFO): implementation and use cases
- Trees: binary trees, binary search trees, traversal algorithms
- Sorting algorithms: bubble, insertion, selection, merge, quicksort — time complexity comparison
- Searching algorithms: linear search, binary search
- Big O notation: understanding time and space complexity

## Key Skills Developed

- Choosing the right data structure for a given problem
- Understanding why certain operations are fast or slow depending on data organization
- Reading and reasoning about algorithmic complexity
- Applying sorting and searching logic in scripting and automation

## How This Applies to My Career

Data structures knowledge is foundational for understanding how the tools IT professionals use actually work under the hood. Monitoring platforms, SIEM systems, and automation tools all process and store data using these structures — understanding them makes it easier to predict performance, diagnose slowness, and interpret how data flows through a system.

In PowerShell scripting, understanding arrays and hash tables (dictionaries) is essential for working with bulk data — iterating through user lists, mapping device configurations, or processing CSV exports from ticketing systems. The algorithmic thinking from this course also helps when optimizing scripts that run against large datasets.
