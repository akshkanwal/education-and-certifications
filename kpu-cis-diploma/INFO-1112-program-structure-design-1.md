# INFO-1112 — Program Structure & Design 1

**Program:** KPU Computer Information Systems Diploma  
**Topic Area:** Programming Logic & Structured Design

## What This Course Covered

- Foundational programming logic and algorithm design
- Pseudocode writing and flowchart creation
- Structured and modular programming principles
- Control structures: sequence, selection (if/else, switch), iteration (for, while, do-while)
- Variable types, scope, and data handling

## Key Skills Developed

- Breaking a problem into logical, repeatable steps before writing any code
- Designing modular solutions where each function does one thing clearly
- Reading and tracing through logic to identify errors
- Translating a real-world task into a structured procedure

## How This Applies to My Career

This course built the thinking pattern behind every PowerShell script I have written in IT environments. Whether automating user account creation in Active Directory, scripting bulk printer deployments, or building a process for onboarding new endpoints, the structured design approach from INFO-1112 is what turns a one-off task into a repeatable, documented solution.

At **ESW IT**, this mindset shows up in how I approach tickets that benefit from scripted resolution versus manual steps, and in the way I document solutions so they can be reused across the team.
