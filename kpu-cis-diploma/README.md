# KPU Computer Information Systems Diploma

**Institution:** Kwantlen Polytechnic University (KPU), Surrey, BC  
**Program:** Computer Information Systems (CIS)  
**Format:** 2-Year Diploma  
**Completed:** 2021

## Program Overview

The CIS Diploma at KPU provides a broad, practical foundation in IT — spanning hardware, networking, programming, database systems, operating systems, security, and project management. The program balances theoretical understanding with hands-on lab work designed to prepare graduates for real enterprise IT environments.

## How This Foundation Shows Up in My Work

Every course in this program has a direct line to something I have done professionally. The networking courses built the foundation for VLAN design, firewall management, and VPN configuration. The server OS course translates directly into Active Directory and Group Policy administration. The systems analysis and project courses provided the structure I used in the F12 project coordinator role.
